# vuepress-plugin-qihu-autopush

> 360 autopush SEO plugin for vuepress
> 奇虎 360 站点自动推送 Vuepress 插件


## Usage

- Install

  ```sh
  npm install -D vuepress-plugin-qihu-autopush
  ```

-  Update `plugins` in `.vuepress/config.js`

  ```js
  module.exports = {
    plugins: [
      'vuepress-plugin-qihu-autopush'
    ]
  };
  ```
  or

  ```js
  module.exports = {
    plugins: {
      'vuepress-plugin-qihu-autopush':{}
    }
  };
  ```
