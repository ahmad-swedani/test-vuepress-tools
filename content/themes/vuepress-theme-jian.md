# vuepress-theme-jian

Just a vuepress theme, Zero config.

## How to Use

#### 1. Install

```
yarn add vuepress-theme-jian
# npm install vuepress-theme-jian
```

#### 2. Using the theme

```js
// .vuepress/config.js
module.exports = {
  theme: 'jian',
}
```
