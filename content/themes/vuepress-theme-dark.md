<h1 align="center">vuepress-theme-dark</h1>
<p align="center">Dark theme for VuePress to reduce eye strain when reading</p>

## Installation
``` bash
npm i -D vuepress-theme-dark

yarn add --dev vuepress-theme-dark
```

## Configuration
``` js
module.exports = {
  theme: 'dark',
};
```
